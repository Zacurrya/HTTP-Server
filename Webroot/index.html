Java HTTP Server Project

About

This is a custom HTTP server implementation in Java that handles basic HTTP requests and responses.

Features

  • HTTP Request Parsing
  • Configuration Management
  • Multi-threaded Server Implementation
  • Support for Headers and Request Methods

Implementation Details

The server includes:

  • HTTP Parser for processing incoming requests
  • Configuration management through JSON files
  • Server listener thread for handling connections
  • Logging support using SLF4J
